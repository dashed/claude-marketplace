# Changelog - long-form-math

All notable changes to the long-form-math skill in this marketplace will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-03-01

### Added
- Initial addition to marketplace
- SKILL.md with core philosophy, three-phase proof workflow, exposition patterns, writing rules, tone guide, and anti-patterns
- references/proof-writing-guide.md with proof templates for 8 techniques (direct, contrapositive, contradiction, cases, existence, uniqueness, induction, epsilon-delta)
- references/proof-writing-guide.md with the almost-correct proof pattern, common mistakes, and technique selection flowchart
- references/exposition-patterns.md with motivation-first patterns, definition introduction workflow, historical storytelling integration
- references/exposition-patterns.md with all 10 Chartrand symbol-usage rules, language guidelines, and equation formatting conventions
- Inspired by Jay Cummings' "Real Analysis: A Long-Form Textbook" and Chartrand's "Mathematical Proofs: A Transition to Advanced Mathematics" (4th ed)
