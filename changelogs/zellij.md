# Changelog - zellij

All notable changes to the zellij skill in this marketplace will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-12-02

### Added
- Initial addition to marketplace
- Comprehensive SKILL.md covering session management, programmatic control, and common workflows
- references/actions.md with complete action reference (50+ actions documented)
- references/layouts.md with KDL layout syntax and examples
- Session management workflows (list, attach, kill, delete)
- Programmatic control via `zellij action` commands (write-chars, dump-screen)
- Common workflows for Python REPL, debugging (gdb/lldb), and interactive git
- Input modes documentation (normal, locked, pane, tab, resize, scroll, session)
- Pane and tab management commands
- Layout system documentation with templates and built-in layouts
