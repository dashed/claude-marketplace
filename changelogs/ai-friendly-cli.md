# Changelog - ai-friendly-cli

All notable changes to the ai-friendly-cli skill in this marketplace will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-03-05

### Added
- Initial addition to marketplace
- Comprehensive skill for making CLIs AI-agent-friendly
- 8 core principles: structured JSON I/O, schema introspection, context window discipline, input hardening, safety rails, structured errors, agent documentation, multi-surface architecture
- Implementation priority checklist
- Reference documentation: output patterns, input hardening, safety patterns, architecture
- Based on "Rewrite your CLI for AI Agents" by Justin Poehnelt and Google Workspace CLI reference implementation
