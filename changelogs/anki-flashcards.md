# Changelog - anki-flashcards

All notable changes to the anki-flashcards skill in this marketplace will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2026-05-14

### Added
- Initial addition to marketplace
- SKILL.md with AnkiConnect installation guidance, connectivity checks, and core workflows
- Comprehensive API reference covering all 100+ AnkiConnect actions across 8 categories
- Flashcard design best practices guide with effective card patterns and anti-patterns
- Support for Basic, Cloze, and custom note types
- Batch operations via multi action
- Media attachment support (images, audio, video)
- Search query syntax reference for finding and filtering cards
