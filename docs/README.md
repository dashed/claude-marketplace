# Marketplace Documentation

Marketplace-level documentation that applies to the entire claude-marketplace project.

## Purpose

This directory contains documentation about the marketplace itself, not individual plugins:

- **Testing strategies** - How the marketplace is tested
- **Architecture** - Overall marketplace design and structure
- **Development guides** - How to work with the marketplace
- **Policies** - Contribution guidelines, versioning policies

## Contents

- [TESTING_STRATEGY.md](./TESTING_STRATEGY.md) - Marketplace testing approach and strategy

## Scope

**Marketplace-level only** - Documentation here applies to the entire project.

For **plugin-specific** documentation, see:
- [notes/](../notes/) - Meta-documentation about individual plugins
- [changelogs/](../changelogs/) - Plugin version history

For **general concepts**, see:
- [references/](../references/) - General reference material about agent skills, plugins, etc.

## Format

All documentation is in Markdown format, following [CommonMark](https://commonmark.org/) specification.
